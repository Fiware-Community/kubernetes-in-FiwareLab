Kubernetes Support

Fiware Lab Kubernetes support provides Platform as a service (PaaS) for Fiware GEs. It orchestrate Fiware GE's on top Kubernetes using key technologies Ansible, Docker, Python Flask etc. It provides Highly available Fiware GE's cluster with versioning feature.

Key features of Fiware LAB Kubernetes Support:

  • High Availability of Fiware GE's.
  • Independence of selecting GE version.
  • Add Fiware GE's/components to existing cluster.
  • Real time log facility of cluster deployment.
